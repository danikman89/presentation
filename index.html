What Electron is?

  • Electron is open-source software framework
  • Make Desktop software with HTML CSS & JavaScript
  • Combines Chromium and Node.js into a single runtime
  • Electron can use React, Angular or Vue for Desktop Apps
  • Apps can be packaged for Windows, Mac & Linux

Apps built on Electron

For more: https://www.electronjs.org/apps

Main process & Render process

  • Main Process - creates Browser Window instances to run web pages
  • Render Process - each webpage runs its own isolated process called the render process

Advantages of using Electron

  • Work on all platforms, including Linux
  • Low barrier to entry
  • A large community of developers and users
  • Automatic updates

Disadvantages of using Electron

  • Each application comes with an entire runtime environment
  • Some applications are of poor quality
  • Large download sizes
  • High memory usage
